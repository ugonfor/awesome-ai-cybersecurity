## SOC / Incident Response

### [Microsoft Security Copilot](https://www.microsoft.com/en-us/security/business/ai-machine-learning/microsoft-copilot-security)
- **Created by**: Microsoft
- **Open Source**: No (commercial, included in Microsoft 365 E5)
- **Summary**: AI-powered security analyst embedded in Defender, Entra, Intune, Purview; 37+ agents with 40+ new agents announced at Ignite 2025; Phishing Triage Agent improves verdict accuracy by 77%.

### [Charlotte AI](https://www.crowdstrike.com/en-us/platform/charlotte-ai/)
- **Created by**: CrowdStrike
- **Open Source**: No
- **Summary**: AI analyst for Falcon platform; triages detections with >98% accuracy, eliminating 40+ hours/week of manual work; Charlotte Agentic SOAR orchestrates AI agents across security lifecycle.

### [Cortex XSIAM / AgentiX](https://www.paloaltonetworks.com/cortex/cortex-xsiam)
- **Created by**: Palo Alto Networks
- **Open Source**: No
- **Summary**: AI-driven SOC platform trained on 1.2 billion real-world playbook executions; delivers 98% MTTR reduction with 75% less manual work. Surpassed $1B in cumulative bookings in 2025.

### [Dropzone AI](https://www.dropzone.ai)
- **Created by**: Dropzone AI
- **Open Source**: No
- **Summary**: Autonomous AI SOC analyst investigating security alerts in 3-10 minutes vs 30-40 minutes manually; integrates with Splunk, CrowdStrike, Microsoft Defender, AWS Security Hub.

### [Exaforce](https://www.exaforce.com)
- **Created by**: Exaforce
- **Open Source**: No
- **Summary**: Full-lifecycle AI SOC platform using task-specific "Exabots"; multi-model AI blending semantic understanding with behavioral analytics; $75M Series A from Khosla Ventures.

### [Radiant Security](https://radiantsecurity.ai)
- **Created by**: Radiant Security
- **Open Source**: No
- **Summary**: Agentic AI SOC platform automating alert triage, investigation, and response; auto-closes false positives and escalates real threats with root cause analysis.

### [Simbian AI](https://simbian.ai)
- **Created by**: Simbian
- **Open Source**: No
- **Summary**: Four autonomous agents (SOC Agent, Threat Hunt Agent, etc.); Microsoft Sentinel integration; 15-fold customer growth in 2025; #1 in AI SOC ARR.

### [Arctic Wolf Cipher](https://arcticwolf.com/resources/press-releases/arctic-wolf-introduces-an-ai-security-assistant-built-on-the-arctic-wolf-aurora-platform/)

## Malware Analysis

### [Microsoft Project Ire](https://www.microsoft.com/en-us/research/blog/project-ire-autonomously-identifies-malware-at-scale/)
- **Created by**: Microsoft Research
- **Open Source**: No (internal)
- **Summary**: Autonomous AI agent for malware reverse engineering using LLMs with angr and Ghidra; precision 0.98, recall 0.83; first machine at Microsoft to author a malware conviction case for an APT sample. Black Hat USA 2025.

### [Joe Reverser](https://www.joesecurity.org/joe-reverser)
- **Created by**: Joe Security
- **Open Source**: No (commercial)
- **Summary**: Agentic automated malware and phishing analyst; autonomously selects reversing tools for static/dynamic analysis; produces reports with IOCs, MITRE ATT&CK mappings, and verdicts.
