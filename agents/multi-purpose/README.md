# Multi-Purpose Cybersecurity AI Agents

> AI agents and frameworks spanning both offensive and defensive cybersecurity tasks.

---

### [Fabric](https://github.com/danielmiessler/fabric)
- **Created by**: Daniel Miessler
- **Open Source**: Yes
- **Summary**: Open-source framework with 200+ specialized prompt patterns for cybersecurity tasks including threat modeling, vulnerability analysis, and security report generation.

### [Floki / Dapr Agents](https://github.com/Cyb3rWard0g/floki)
- **Created by**: Roberto Rodriguez (Cyb3rWard0g)
- **Open Source**: Yes
- **Summary**: Framework for building LLM-based autonomous agents with Dapr; supports event-driven multi-agent collaboration and persistent state; donated to CNCF Dapr project (released as Dapr Agents March 2025).

### [BurpGPT](https://github.com/aress31/burpgpt)
- **Created by**: aress31
- **Open Source**: Yes
- **Summary**: Burp Suite extension integrating LLMs (OpenAI, Anthropic, Gemini, Ollama) for additional passive scanning to discover bespoke vulnerabilities.

### [Burp AI Agent](https://github.com/six2dez/burp-ai-agent)
- **Created by**: six2dez
- **Open Source**: Yes
- **Summary**: Burp Suite extension with 7 AI backends, 53+ MCP tools, and 62 vulnerability classes across passive and active AI scanners.
